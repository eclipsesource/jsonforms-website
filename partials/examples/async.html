The uischema, schema, and data attributes of the jsonforms expect values that represent the UI schema, the JSON schema, and the actual data to be rendered, respectively. But the values do not necessarily need to be present when rendering is triggered. It is also possible to pass promises or functions returning promises to the directive.

This is demonstrated in the following way: You can load data, schema and UI in an arbitrary order, as soon as everything is available the UI is rendered.
